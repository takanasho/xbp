第6回　プログラミング

私は、プログラミングでストップウォッチができるのではないかと考えました。

コード

  1. import time
  4. def convert(sec):
  5.     minits = sec // 60
  6.     second = sec % 60
  7.     milli_sec = (second - int(second)) * 1000
  8.     hour = minits // 60
  9.     min = minits % 60
  10.     return f"{int(hour)}時{int(min)}分{int(second)}秒{int(milli_sec)}"
  12. print("★ストップウォッチ★")
  13. start_signal = input('”ENTER”で開始')
  14. start_time = time.time()
  16. print('"ENTER"でラップ')
  17. print('”p”と"ENTER"で停止')
  18. lap_num = 0
  19. while True:
  20.     stop_signal = input()
  21.     stop_time = time.time()
  22.     if stop_signal == '':
  23.         lap_num += 1
  24.         result = stop_time - start_time
  25.         time_result = convert(result)
  26.         print(f"lap {lap_num}：{time_result}")
  27.     elif stop_signal == 'p':
  28.         result = stop_time - start_time
  29.         time_result = convert(result)
  30.         print(f"停止時間"f"：{time_result}")
  31.         break
  32.     else:
  33.         print('"ENTER"でラップ')
  34.         print('”p”と"ENTER"で停止')#ここで入力の違いを伝える

機能

こちらはストップウォッチの機能として0から時間がはかれるとともにラップタイムも図ることができます。
